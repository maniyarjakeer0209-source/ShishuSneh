Shishu Sneh – Smart Child Healthcare Monitoring System
Overview

Shishu Sneh is a smart healthcare monitoring application designed to assist parents and healthcare providers in managing child healthcare activities efficiently. The application focuses on vaccination tracking, nutrition monitoring, growth analysis, healthcare reminders, and cloud-based healthcare data management.

The project was developed using modern Android development technologies along with Firebase and Google Cloud integration to provide secure, scalable, and real-time healthcare monitoring workflows.
